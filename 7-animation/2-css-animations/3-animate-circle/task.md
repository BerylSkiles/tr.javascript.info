# Animasyonlu Daire

`showCircle(cx, cy, radius)` şeklinde, animasyonlu bir şekilde daireyi genişleten, bir fonksiyon yarat.

- `cx,cy` dairenin merkezinin pencereye göreceli olarak koordinatları,
- `radius` dairenin yarıçapı.
