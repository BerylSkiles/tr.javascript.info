# `if..else`'i `'?'` formunda yazınız

Aşağıdaki `if..else` cümlesini "üçlü" operatörler `'?`' olarak yazın:

Okunabilirlik açısından kodu satırlara bölmeniz önerilir.

```js
let mesaj;

if (giris == 'Çalışan') {
  mesaj = 'Merhaba';
} else if (giris == 'Yönetici') {
  mesaj = 'Sana da merhaba patron';
} else if (giris == '') {
  mesaj = 'Giriş yasak';
} else {
  mesaj = '';
}
```
