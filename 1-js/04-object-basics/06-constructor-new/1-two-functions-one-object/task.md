# İki fonksiyon - bir obje

`new A() == new B()` şeklinde `A` ve `B` fonksiyonları yaratmak mümkün müdür?

```js no-beautify
function A() { ... }
function B() { ... }

let a = new A;
let b = new B;

alert( a == b ); // true
```
